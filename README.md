# Stroke Prediction 中風預測

使用 [kaggle 中風預測資料集](https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset) 做資料分析，流程包含資料探索、處理缺失值、特徵工程、移除不需要的欄位、特徵縮放...等，  
再將資料集訓練六種分類模型，最後建立頁面讓使用者輸入數值進行中風預測。
